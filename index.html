Here is My Software Engineering Portfolio

LazyDate

LazyDate is a full-stack application that takes a user’s interests and location and generates personalized date ideas. It was also a collaborative effort between me and 3 other software. On the backend, Mongoose ODM is used to handle multiple collections in a scalable and flexible way. The crowning jewel of this project is how we integrated our app with Yelp’s API, using a sleek user interface.

Wander

Wander is a content-driven personal blog site, where sharing anything and everything is not only allowed but encouraged. Inspired by tumblr, Wander allows users to express themselves, connect with other likeminded users, whilst joining new communities. Wander is created with React/Redux on the front-end, Rails/PostgresSQL on the back-end and it utilizes AWS S3 for Cloud Storage.

ALONE

ALONE is a visual experiment that explores the relationship between lonelieness, technology, isolation, and the need for connection. Inspired by quarantine, it marries audio, poetry, and technology to explore these themes. It is a multi-page web application that harnesses the power of lightweight Javascript libraries like Three.js, Bootstrap JS and Particle.js, to create cinematic animated scenes.

Technologies that I am proficient in include: Ruby, Ruby on Rails, Mongo DB, Javascript, node.js, Express, CSS, HTML, SASS, and SQL.
